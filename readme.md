Reddit Improvement Zzuite (RIZz)

The goal of this extension is to enhance/improve Reddit by giving users tools to curate their own experience on the site. The name RIZz is a loving homage to Reddit Enhancement Suite (RES).

RIZz supports both the old and current Reddit designs.

Enhancements/Improvements provided by this extension include:

\- Advanced post filtering (Regex)  
\- UI tweaks and style/layout customizations  

Possible future additions:

\- User tagging  
\- Topic filters  
\- Old Reddit clone mode  
\- More UI tweaks
